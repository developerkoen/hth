Uitleg over <form> in HTML

Wat is <form>?

De <form>-tag wordt gebruikt om een formulier op een webpagina te maken. Hiermee kunnen gebruikers gegevens invoeren en deze naar een server verzenden.

Voorbeeld

Hier is een voorbeeld van een eenvoudig formulier:

De HTML-code

De code voor het bovenstaande formulier ziet er zo uit:

                
<form action="https://example.com/submit" method="post">
    <label for="name">Naam:</label>
    <input type="text" id="name" name="name" placeholder="Voer je naam in">
    
    <label for="email">E-mailadres:</label>
    <input type="email" id="email" name="email" placeholder="Voer je e-mailadres in">
    
    <button type="submit">Verzenden</button>
</form>
