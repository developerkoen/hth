Uitleg over <audio> in HTML

Wat is <audio>?

De <audio>-tag wordt gebruikt om een audiobestand af te spelen op een webpagina. Dit is handig voor muziek, podcasts of geluidseffecten.

De HTML-code

De code voor het bovenstaande audiobestand ziet er zo uit:

                
<audio controls>
    <source src="https://www.w3schools.com/html/horse.ogg" type="audio/ogg">
    <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
    Je browser ondersteunt dit audioformaat niet.
</audio>
