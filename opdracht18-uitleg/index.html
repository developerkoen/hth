Uitleg over media queries in CSS

Wat is een media query?

Een media query wordt gebruikt in CSS om de stijl van een element aan te passen op basis van de schermgrootte. Dit helpt bij het maken van een responsieve website die goed werkt op zowel grote als kleine schermen.

Voorbeeld

Probeer het scherm te verkleinen om de kleur te veranderen!

De HTML- en CSS-code

De code voor de bovenstaande media query ziet er zo uit:

                
<!-- HTML -->
<div class="responsive-box">
    Probeer het scherm te verkleinen om de kleur te veranderen!
</div>

<!-- CSS -->
.responsive-box {
    width: 80%;
    max-width: 600px;
    padding: 20px;
    text-align: center;
    background-color: #4A90E2; /* Blauw */
    color: #FFFFFF; /* Witte tekst */
    border-radius: 10px;
    margin: 0 auto;
}

/* Media query voor kleinere schermen */
@media (max-width: 600px) {
    .responsive-box {
        background-color: #50C878; /* Groen */
        color: #1F2937; /* Donkergrijze tekstkleur */
    }
}
