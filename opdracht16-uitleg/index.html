Uitleg over scrollbare inhoud in HTML

Wat is scrollbare inhoud?

Met CSS kun je een element scrollbaar maken door gebruik te maken van de overflow-eigenschap. Dit is handig voor content die niet in een vaste ruimte past.

De HTML- en CSS-code

De code voor de bovenstaande scrollbare inhoud ziet er zo uit:

                
<!-- HTML -->
<div class="scroll-box">
    Dit is een voorbeeld van een div met scrollbare inhoud.<br><br>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus euismod eros sed nunc tincidunt, nec vulputate risus efficitur.<br><br>
    Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium.<br><br>
    At vero eos et accusamus et iusto odio dignissimos ducimus qui blanditiis praesentium voluptatum deleniti atque.
</div>

<!-- CSS -->
.scroll-box {
    width: 80%;
    max-width: 500px;
    height: 150px;
    overflow: auto;
    background-color: #FFFFFF;
    color: #1F2937;
    border: 2px solid #4A90E2;
    padding: 10px;
    margin: 0 auto;
    text-align: left;
}
