HTML Cursus Plan

🌍 Hoofdstructuur

  • 📂 Startscherm met opdrachtenkeuze
  • 📊 Voortgangsbalk met percentage voltooid
  • 📝 Opdrachtensysteem met interactief leren

🏗 Structuur van een Opdracht

Elke cursus bevat een variabel aantal opdrachten, afhankelijk van de benodigde leerstof. Elke opdracht bevat de volgende onderdelen:

  1. 📖 Theorie & uitleg
  2. ❓ Korte quiz
  3. 🎮 Speelse interactieve vraag
  4. 🎥 Video met uitleg
  5. 💡 Toelichting op de volgende opdracht
  6. 🛠 Zelf aan de slag

🔗 Verbindingen tussen opdrachten

Flow van opdrachten:

Startscherm → Opdracht 1 → Opdracht 2 → Opdracht 3 → ... → Opdracht 20 → Eindopdracht

Binnen een opdracht:

Theorie → Quiz → Interactieve Vraag → Video → Toelichting → Opdracht → Voltooid

📌 Extra Functies

  • 🎖 Badges voor voortgang
  • 📁 Opslaan in browser met localStorage
  • 📊 Overzichtspagina met voltooide opdrachten
