Uitleg over achtergrondkleur in HTML

Wat is een achtergrondkleur?

Met CSS kun je eenvoudig een achtergrondkleur toevoegen aan een HTML-element met de eigenschap background-color.

Voorbeeld

Hier is een voorbeeld van een element met een achtergrondkleur:

Dit is een div met een gouden achtergrondkleur!

De HTML- en CSS-code

De code voor het bovenstaande voorbeeld ziet er zo uit:

                
<div class="colored-box">
    Dit is een div met een gouden achtergrondkleur!
</div>

<style>
    .colored-box {
        background-color: #FFD700; /* Goudgele achtergrond */
        padding: 20px;
        text-align: center;
        font-weight: bold;
        border-radius: 5px;
        color: #1F2937; /* Donkergrijze tekstkleur */
    }
</style>
