Uitleg over <ul> en <ol> in HTML

Wat zijn <ul> en <ol>?

Met de <ul>- en <ol>-tags kun je lijsten maken in HTML:

  • <ul>: Een ongeordende lijst (bullets).
  • <ol>: Een geordende lijst (genummerd).

Voorbeeld

Hier zijn voorbeelden van een ongeordende en een geordende lijst:

Ongeordende lijst (<ul>)

  • Appels
  • Bananen
  • Kersen

Geordende lijst (<ol>)

  1. HTML leren
  2. CSS leren
  3. JavaScript leren

De HTML-code

De code voor de bovenstaande lijsten ziet er zo uit:

                
<!-- Ongeordende lijst -->
<ul>
    <li>Appels</li>
    <li>Bananen</li>
    <li>Kersen</li>
</ul>

<!-- Geordende lijst -->
<ol>
    <li>HTML leren</li>
    <li>CSS leren</li>
    <li>JavaScript leren</li>
</ol>
