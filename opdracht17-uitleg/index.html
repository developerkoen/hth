Uitleg over een knop (<button>) in HTML

Wat is een knop?

Met de <button>-tag kun je een klikbare knop maken. Dit kan gebruikt worden voor acties zoals het verzenden van formulieren, het uitvoeren van scripts, of het navigeren naar een andere pagina.

De HTML- en CSS-code

De code voor de bovenstaande knop ziet er zo uit:

                
<!-- HTML -->
<button onclick="alert('Je hebt op de knop geklikt!')">Klik hier</button>

<!-- CSS -->
button {
    background-color: #4A90E2; /* Blauw */
    color: #FFFFFF; /* Witte tekst */
    border: none;
    padding: 10px 20px;
    font-size: 16px;
    border-radius: 5px;
    cursor: pointer;
    transition: 0.3s;
}

button:hover {
    background-color: #50C878; /* Groen */
}
