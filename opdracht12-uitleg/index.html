Uitleg over <a href="mailto:"> in HTML

Wat is <a href="mailto:">?

Met de <a href="mailto:">-tag kun je een klikbare link maken waarmee bezoekers direct een e-mail kunnen sturen naar een specifiek e-mailadres.
