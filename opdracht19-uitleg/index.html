Uitleg over een favicon in HTML

Wat is een favicon?

Een favicon is een klein pictogram dat wordt weergegeven in het tabblad van je browser. Dit helpt bij herkenbaarheid van een website.

Voorbeeld

Als je naar de tab van deze pagina kijkt, zie je een favicon!

De HTML-code

De code om een favicon toe te voegen ziet er zo uit:

                
<link rel="icon" type="image/png" href="https://www.w3schools.com/html/img_girl.jpg">
