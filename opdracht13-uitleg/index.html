Uitleg over tekstkleur in HTML

Wat is kleur in tekst?

In CSS kun je de kleur van tekst aanpassen met de eigenschap color. Je kunt verschillende kleurmodellen gebruiken zoals namen, hex-codes of RGB-waarden.

Voorbeeld

Hier zijn voorbeelden van tekst met verschillende kleuren:

Dit is blauwe tekst.

Dit is rode tekst.

Dit is groene tekst.

De HTML- en CSS-code

De code voor de bovenstaande gekleurde tekst ziet er zo uit:

                
<!-- HTML -->
<p class="blue-text">Dit is blauwe tekst.</p>
<p class="red-text">Dit is rode tekst.</p>
<p class="green-text">Dit is groene tekst.</p>

<!-- CSS -->
.blue-text {
    color: #4A90E2; /* Blauw */
    font-weight: bold;
}

.red-text {
    color: red;
    font-weight: bold;
}

.green-text {
    color: #50C878; /* Groen */
    font-weight: bold;
}
