Uitleg over <a> in HTML

Wat is <a>?

De <a>-tag wordt gebruikt om een hyperlink toe te voegen aan je webpagina. Hiermee kun je gebruikers naar een andere pagina, een andere website of een specifiek onderdeel op dezelfde pagina leiden.

Voorbeeld

Hier is een voorbeeld van een hyperlink:

Ga naar Google

De HTML-code

De code voor de bovenstaande hyperlink ziet er zo uit:

                
<a href="https://www.google.com" target="_blank">Ga naar Google</a>
